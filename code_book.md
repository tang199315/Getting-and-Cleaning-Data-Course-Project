# Code Book
This code book summarises the data fields in the data set   `preprocessed_data_set.txt`.

## Data Fields
Data fields can be devided into two parts, identifiers and feature measurements from the 3-axial accelerometer and gyroscope. Please refer to `feature_info.txt` for detailed specifications of feature measurements.

 
### Identifiers
* `subject` : Identity for volunteer who performed the activity. Its range is from 1 to 30
* `activity` : Activity performed in the experiment. Its value must be one of the following:
    * WALKING
    * WALKING_UPSTAIRS
    * WALKING_DOWNSTAIRS
    * SITTING
    * STANDING
    * LAYING
